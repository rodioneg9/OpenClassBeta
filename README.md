# OpenClassBeta

Desktop client for Google Classroom with AI features.

## Features
- Focus Mode
- Assignment View
- AI Assistant
- Digital Notebook

## Roadmap

- [x] Basic Classroom client
- [x] Focus Mode
- [ ] Backend auth
- [ ] AI assistant
- [ ] Plugin system
      
## Setup

1. Install dependencies
npm install

2. Run
npm run dev

## OAuth setup

Create OAuth Desktop App in Google Cloud and set:

GOOGLE_CLIENT_ID=your_id

GOOGLE_CLIENT_SECRET=your_secret
## Build

npm run dist
